# This is a project about modus and pandas

## Library:
- pandas
- pymodbus
- multiprocessing
- dotenv

## Install:
- python3 -m venv my_venv
- source my_venv/bin/activate
- pip3 install pandas
- pip3 install pymodbus
- pip3 install multiprocess
- pip3 install python-dotenv

## Run:
python3 run_modbus.py
